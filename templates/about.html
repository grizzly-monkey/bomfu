About BomFu

BomFu is the kung-fu of Bill of Materials. It's both a website and an open file format. The idea is BOMs can be beautifully shared here but also created offline, emailed, and script-generated.

BomFu solves typical BOM requirements like providing multiple views to the same data, e.g. order/build views. It also tackles more advanced challenges like alternative supplier ranking and multi-country, multi-currency support.

The file format is completely human editable and you are encouraged to write your BOMs in a plain text editor. This makes it compatible to git-style versioning and storing right with your hardware projects.

The file format is also very easy to generate and parse. This allows for deriving BOMs from CAD files and other primary sources.
